Hostile entity alarm support:
- Wardens and withers are detected through any peripheral exposing an entity scan method such as getEntitiesInRange, getNearbyEntities, scanEntities, getEntities, getMobs, getLivingEntities, or sense.
- If an extension uses a different method name, configure the sensor with type = "entity", peripheral = "<name>", and method = "<method>".
